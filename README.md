# devpod-go-1.24

DevPod configuration for Go 1.24 development environment.

## What's included

- Go 1.24 base image (`build-harbor.alauda.cn/devcontainers/go:1.24`)
- Claude Code CLI
- kubectl
- tmux
- Go proxy configured for Alauda internal repos

## Usage

Use this repo as a DevPod workspace source:

```bash
devpod up github.com/kycheng/devpod-go-1.24
```
